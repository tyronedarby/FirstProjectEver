# Candidate Refund System (High School SBA)

## 📌 Project Overview

This is a legacy project and the very first program I wrote in high school (Grade 11, 2020). It was developed as part of my **Information Technology School Based Assessment (SBA)**.

The program is written in **Pascal** and serves as a basic election data processor. It calculates whether a political candidate is eligible for a deposit refund based on the percentage of total votes they received in their constituency.

## ⚙️ Features

* **Data Entry:** Accepts input for constituency name, candidate name, total votes cast, and votes received.
* **Logic Processing:** Implements a $20\%$ threshold ($VR \ge 0.2 \times VC$) to determine refund eligibility.
* **Looping:** Iterates through 10 entries using a `While` loop.
* **Summary Reporting:** Displays a final count of all candidates who qualified for a refund.

## 🛠️ Technologies Used

* **Language:** Pascal
* **Year:** 2020
* **Purpose:** Academic / High School SBA

## 🚀 How to Run

To run this code today, you will need a Pascal compiler such as **Free Pascal** or an online compiler.

1. Copy the code from `sba.pas`.
2. Compile using a Pascal compiler:
```bash
fpc sba.pas

```


3. Run the executable:
```bash
./sba

```
